What is the network?
A network is a collection of devices connected to each other to allow the sharing of data. Example of a network is an internet. An internet connects the millions of people across the world.
What are the advantages of Distributed Processing?
A list of advantages of distributed processing:
  • Secure
  • Support Encapsulation
  • Distributed database
  • Faster Problem solving
  • Security through redundancy
  • Collaborative Processing
Which are the different factors that affect the performance of a network?
The following factors affect the performance of a network:
  • Large number of users
  • Transmission medium types
  • Hardware
  • Software
