What is operating system?
It is a program that acts as an Intermediary between the user and the computer hardware. Purpose :
  • It provide convenient environment in which user can execute programs in efficient manner.
  • It is a resource allocator responsible for allocating system resources.
  • It control program which controls the operation of computer hardware.
What are the components of computer system?
Hardware,Os,Application programs,the users.
What are the types of operating system?
Batched operating systems : The user of batch operating system never directly interacts with the computer. Time_sharing Operating Systems : It enables people located at different terminal to use a single computer system at the same time. Real time OS : Time interval to process and responds to input is very small. Ex : Space software and military software Distributed OS : It uses many processors located in different machines to provide very fast computations. Multiprogramming OS : OS keeps several jobs in memory at a time-sharing the processor,when two or more programs reside in memory at the same time.
